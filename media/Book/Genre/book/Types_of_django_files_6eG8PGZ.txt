manage.py - для управления проектом, а также выполнение команд с его помощью

wsgi.py и asgi.py - нужны для асинхронных веб-серверов и приложений 

urls - файл для url проекта, нужен для поключения url приложений 

settings.py - файл для работы со всеми настройками проекта

__init__.py - модуль пайтон

admin.py -  для регистрации моделей и вывода в админ панель, также для написания логики для изменения поведения админки

apps - чтобы настроить любую конфигурацию для приложения, а также для настройки некоторых атрибутов приложения

models.py -  для написания моделей и описания таблиц  в БД

tests.py - для написания тесов для приложения 

views.py - логика приложения, прописание функций или классой которые принимают запрос и возвращают 

папка migrations - для хранения файлов миграции нашего приложения

forms.py - для прописания форм
